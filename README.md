# math-to-code-ml

『統計的機械学習の数理』を読み解き、数式の霧を Python コードで晴らしていくための学習ノートです。

## 📖 コンセプト
数物系の専門書特有の「厳密すぎる記号」に心を折られないための、自分による自分のための翻訳リポジトリ。
- **数式をコードへ**: $\mathbf{w}^T \mathbf{x}$ と書かれたら `w.T @ x` と書く。
- **抽象を具現へ**: 定義を読んだら、具体的な数値を入れて `print()` する。
- **継続を習慣へ**: GitHub の草を生やしながら、一歩ずつ進む。

## 📑 記号・概念の翻訳辞書 (随時更新)
「これは型宣言である」と割り切るためのカンニングペーパーです。

| 数理記号 | 読み方 | 意味 | Python / NumPy でのイメージ |
| :--- | :--- | :--- | :--- |
| $a \in A$ | 属する | $a$ は集合 $A$ の要素である | `a in list_A` (型チェックに近い) |
| $\mathbb{R}$ | 実数 | 虚数じゃない普通の数すべて | `float` |
| $\mathbb{R}^n$ | $n$次元実ベクトル | 実数が $n$ 個並んだデータ | `np.array([1.0, 2.0, ...])` |
| $\mathbf{x}^T$ | 転置 | 行列やベクトルの行と列を入れ替える | `x.T` |
| $\mathbf{w}^T \mathbf{x}$ | 内積 | 要素ごとの積の合計 | `w.T @ x` または `np.dot(w, x)` |
| $\|\mathbf{x}\|_2$ | L2ノルム | ベクトルの長さ（距離） | `np.linalg.norm(x)` |

## 📁 構成

- `/notebooks`: 実装と可視化の実験場（Jupyter Notebook）
- `/scripts`: 再利用可能なアルゴリズム（Python）
